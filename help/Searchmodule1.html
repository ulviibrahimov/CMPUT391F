After you login, when you click on the Search in the header, it will display a list of search type for the images. Search by Keyword will required you to input the keyword(s). Search by Date will required you to input two date as a time period. Search by Keyword and Date will required you to input the keyword(s) and the time period.

For each type of search, you can choose to order the result images in different way. If you select Recent First, the most recent image will display at first. If you select Recent Last, the oldest image will display at first. If you select Relevant, the most relevant image will display at first. If you do not select the order, the images will display in Recent First model.

For each type of search, you can use the return link at the bottom to return to search type page.

When you fill the inputs in any search type, press the search button to view the searched images.

In the search result page, you can choose any images and see its description.

If you are the author of this image, there will be an edit link that allow you to edit the description of this image.

If you finish reading the descriptions, use the return link at the bottom to return to the search results.

If you use the return in below the search result, you will return to the search type page.
