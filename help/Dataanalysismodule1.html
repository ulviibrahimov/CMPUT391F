If you are the ‘admin’, we provide the sixth module: data analysis module for you.

When you are login as ‘admin’, you can click on Data Analysis use the data analysis module.

In the data analysis module, you need to select a user, and/or enter a subject, and/or provide a Time period. You can also select any sort, and/or the time range. When you finish you input, press the submit to see the result. If you want to return to the input page, use the Return below the result in the Data Analysis Result page.
