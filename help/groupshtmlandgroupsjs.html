Displays two lists by default. The lists displayed are dependent on the user stored in session.


The first list provides the option to create a new group and is followed by a list of groups owned


by the currently logged in user. Each group listed here has corresponding button that will allow


the user to manage/edit the group.


The second list contains a list of all groups that the user belongs to but does not own. It also


provides a 'Leave' button for each group which allows the user to leave the group.


An conditional third list is displayed if the currently logged in user is the 'admin'. This list


contains all the groups that current exist and provides the admin with the option manage any of


these groups.
