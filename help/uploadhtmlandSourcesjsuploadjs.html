Before the page is loaded BuildHeader.js ensures that a user is logged in, if not the browser is


redirected to the login page.


This page is used for uploading new images.


For multiple file uploading the plugin Uploadify (http://www.uploadify.com/) is used.


Before the page is finished loading a call is sent out to the server to get the groups that the user


belongs to. This call ensures the user is logged in again and returns a list of groups. This is


then used to populated the group dropdown for selecting a group.


After the user has a selected an image (or images) and populated the desired fields they can hit


upload. This will send one request per image to the server.


On upload, the inputs are initially validated and processed client side before being sent to the


server. The server then ensures that the user is logged in once more before committing the


new image.
